Namera / Resin Bon - Full Test Cases
Date: 2026-07-21
File: 2026-07-21-full-test.txt
Scope: Complete role-based test matrix for Guest, Customer, and Owner user flows.

Legend:
- AUTOMATED_PASS: covered by the current automated suite and passing at the time this matrix was prepared.
- PLANNED: required test case documented for manual QA or future automation.
- API: backend endpoint/service test.
- UI: frontend page/component flow test.
- E2E: browser/API combined flow.

Current implemented automated suites to keep running:
1. Backend: dotnet test namera_API.Tests\namera_API.Tests.csproj --no-restore
2. Frontend: npm test -- --run
3. Frontend build: npm run build
4. Frontend lint: npm run lint

Core roles:
1. Guest: user is not authenticated.
2. Customer: authenticated user with Customer role.
3. Owner: authenticated user with Owner role.

Reference routes and endpoints:
- Public UI: /, /products, /categories, /about, /contact, /cart, /login, /register
- Customer UI: /customer/dashboard, /account/profile, /account/orders, /account/reviews, /account/addresses
- Owner UI: /owner/dashboard, /owner/products, /owner/products/new, /owner/products/:productId/edit, /owner/categories, /owner/orders, /owner/customers
- Auth API: POST /api/auth/register, POST /api/auth/login, GET /api/auth/me
- Public products API: GET /api/products, GET /api/products/{slug}, GET /api/products/categories
- Customer API: GET/PUT /api/customer/profile, GET /api/customer/dashboard, GET/POST/PUT/DELETE /api/customer/addresses, GET/POST/DELETE /api/customer/reviews
- Customer orders API: GET/POST /api/customer/orders
- Owner products API: GET/POST/PUT /api/admin/products, GET /api/admin/products/{id}, PATCH /api/admin/products/{id}/publish, PATCH /api/admin/products/{id}/archive
- Owner categories API: GET/POST/PUT /api/admin/products/categories, PATCH /api/admin/products/categories/{id}/active
- Owner uploads API: POST /api/admin/products/uploads/product-image, POST /api/admin/products/uploads/category-image
- Owner orders/customers/dashboard API: GET /api/admin/orders, PATCH /api/admin/orders/{id}/status, GET /api/admin/customers, GET /api/admin/dashboard/stats
